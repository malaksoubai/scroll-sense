📕 Have a PDF or Word doc?

Chrome extensions can't read PDF/DOCX binary files directly. Run this one-line command in your Terminal to convert it to a .txt file, then upload that:

PDF → txt:

python3 -m pdfminer.high_level yourfile.pdf > yourfile.txt

DOCX → txt:

python3 -c "import docx; print('\n'.join([p.text for p in docx.Document('yourfile.docx').paragraphs]))" > yourfile.txt

Need the packages first? Run: pip install pdfminer.six python-docx

How it works
