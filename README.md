# ScrollSense — Setup Guide

**AI-powered Chrome extension that redirects you back to your actual work while you scroll Instagram.**

---

## What It Does

ScrollSense watches your Instagram feed and interrupts your scrolling in two alternating ways:

- **Quiz card** — a multiple-choice question generated by Ollama from your own uploaded study files. Answer correctly and earn +2 minutes added to your daily budget.
- **Redirect card** — a bottom-right nudge showing your currently open tabs, so you can jump back to real work in one click.

On top of that:

| State | Trigger | What Happens |
|---|---|---|
| 🟢 **Timer pill** | Always | Shows time remaining in the top-right corner of Instagram |
| ⚠️ **90% warning** | At 90% of budget | Pill shakes and a toast appears warning you a freeze is coming |
| 🛑 **Hard freeze** | At 100% of budget *(optional)* | Feed blurs, all your open tabs appear, Ollama writes a personal message |

---

## Step 1 — Install Ollama + Pull the Model

1. Download and install Ollama from **[ollama.com](https://ollama.com)**
2. Open **Terminal** (Mac/Linux) or **PowerShell** (Windows) and run:

```bash
ollama pull llama3.2
```

This downloads ~2GB. Wait for it to finish.

3. Verify it works:

```bash
ollama run llama3.2 "Say hello in one sentence."
```

You should see a short reply. The AI is ready.

---

## Step 2 — Configure Ollama Origins (Windows — one-time)

By default, Ollama blocks requests from Chrome extensions for security reasons. You need to tell it to accept them. **This is a one-time setup.**

### Permanent fix (recommended)

1. Open the **Start menu** and search for **"Edit the system environment variables"**
2. Click **Environment Variables**
3. Under **User variables**, click **New**
4. Set:
   - **Variable name:** `OLLAMA_ORIGINS`
   - **Variable value:** `*`
5. Click **OK** on all dialogs
6. **Restart your computer**

After this, Ollama will always accept ScrollSense requests automatically — no terminal needed.

### Temporary fix (current session only)

If you don't want to restart, run this in PowerShell each time before using ScrollSense:

```powershell
taskkill /F /IM "ollama app.exe"
taskkill /F /IM ollama.exe
$env:OLLAMA_ORIGINS="*"; ollama serve
```

Keep that terminal window open while using ScrollSense. Closing it stops Ollama.

> **Mac/Linux users:** Add `OLLAMA_ORIGINS=*` to your shell profile (`.zshrc` or `.bashrc`) for the same permanent effect:
> ```bash
> echo 'export OLLAMA_ORIGINS="*"' >> ~/.zshrc
> source ~/.zshrc
> ```

## Step 3 — Load the Extension in Chrome

1. Open Chrome and go to: `chrome://extensions`
2. Toggle **Developer mode** ON (top-right switch)
3. Click **"Load unpacked"**
4. Select the `extension/` folder inside this project
5. **ScrollSense** will appear in your extensions list

---

## Step 4 — Add Study Material (Quiz Me)

1. Click the **ScrollSense icon** in the Chrome toolbar
2. Click **Quiz Me** (opens the options page)
3. Upload a `.txt`, `.md`, `.html`, or `.csv` file — or paste text directly
4. ScrollSense sends the content to Ollama, which generates 20 questions and stores them locally
5. A progress bar shows generation in real time (~1 minute)

Once questions are ready, they'll appear automatically as you scroll Instagram.

> **Have a PDF or Word doc?** Convert it to `.txt` first:
>
> ```bash
> # PDF
> python3 -m pdfminer.high_level yourfile.pdf > yourfile.txt
>
> # DOCX
> python3 -c "import docx; print('\n'.join([p.text for p in docx.Document('yourfile.docx').paragraphs]))" > yourfile.txt
>
> # Install packages if needed
> pip install pdfminer.six python-docx
> ```

---

## Step 5 — Start Scrolling

1. Go to **instagram.com** in Chrome
2. The **timer pill** appears in the top-right (shows time left in your budget)
3. Scroll ~3 screen-lengths and the first intervention fires
4. Every other intervention is a quiz; the alternating ones are redirect cards

**Settings in the popup:**

| Setting | Default | What it controls |
|---|---|---|
| Daily budget | 20 min | Total allowed Instagram time per day |
| Intervene after | 10 | Posts-per-intervention scroll threshold |

## Stats Page

Click **📊 Stats** in the popup (or navigate to it in the options page) to see:

- **All-time summary** — total hours, quiz accuracy, minutes earned, overrides, days tracked
- **7-day bar chart** — green bar = under budget, red = over budget
- **Insights** — peak scroll hour, usage summary, quiz performance
- **Day-by-day history table** — full breakdown per day

## Reset Options

Click **🔄 Reset** in the options page for three levels:

| Level | Wipes | Requires |
|---|---|---|
| Reset Today | Today's timer, scores, overrides | Confirm dialog |
| Reset All History | All daily data + settings (keeps files & questions) | Type `RESET` |
| Full Wipe | Everything — history, settings, files, question bank | Type `DELETE ALL` |

The popup also has a **"Reset today's data"** button for quick mid-day resets.

## How the AI Works

ScrollSense uses Ollama running at `http://localhost:11434` — completely local, free, no API key.

**Question generation** (happens when you upload a file in Quiz Me):
- Ollama reads chunks of your study material
- Generates one multiple-choice question per chunk
- Stores all questions locally in Chrome storage
- No Ollama calls happen while you scroll — questions are served instantly from storage

**Freeze message** (happens when you hit 100% of your budget):
- Ollama writes a short, personal sentence referencing your most recent open tab
- Called from the background service worker (no CORS restrictions)

**Weekly reflection** — After 7 days, Ollama summarizes your usage week in one insight sentence.

If Ollama is unreachable, ScrollSense shows a fallback message and never crashes.

## Troubleshooting

**Timer pill doesn't appear on Instagram**
→ Go to `chrome://extensions` → click **Reload** on ScrollSense → refresh Instagram.

**Ollama dot is red / questions won't generate**
→ Ollama is not running or not configured correctly. Follow Step 2 to set `OLLAMA_ORIGINS=*` permanently. On Windows, confirm by running `ollama serve` in a fresh PowerShell — you should see `Listening on 127.0.0.1:11434`.

**"Only one usage of each socket address" error when running `ollama serve`**
→ Ollama is already running in the background. That's fine — you just can't start a second copy. The real issue is that it started without `OLLAMA_ORIGINS=*`. Fix: kill it first, then restart with the flag (or use the permanent environment variable fix in Step 2 so this never happens again).
```powershell
taskkill /F /IM "ollama app.exe"
taskkill /F /IM ollama.exe
$env:OLLAMA_ORIGINS="*"; ollama serve
```

**0 questions generated after upload**
→ Almost always an `OLLAMA_ORIGINS` issue. See above. Confirm Ollama is running, then try uploading again. You can watch the background service worker console at `chrome://extensions` → ScrollSense → **"service worker"** for detailed logs.

**Pop-ups aren't appearing while scrolling**
→ Make sure you've scrolled downward continuously (~3 screen-lengths triggers the first intervention). Upward scrolling doesn't count.

**Extension won't load in Chrome**
→ Make sure you selected the `extension/` folder (the one containing `manifest.json`), not the parent `scrollsense/` folder.

**Hard freeze override button doesn't work**
→ Reload the extension at `chrome://extensions` and refresh Instagram. This can happen if the extension was updated without reloading.

## Changing the AI Model

Pull a different model (e.g. `ollama pull mistral`), then update line 6 of `background.js`:

```js
const OLLAMA_MODEL = "mistral";
```

Reload the extension at `chrome://extensions`.
